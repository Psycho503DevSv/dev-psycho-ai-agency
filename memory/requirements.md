# Requisitos Globales (Plantilla / Placeholder)

Este archivo sirve únicamente como plantilla o referencia global del sistema.

> [!WARNING]
> **Aislamiento por Proyecto:** Los requisitos de un proyecto específico NO deben escribirse en este archivo global. Deben residir en:
> `memory/projects/{nombre_del_proyecto}/requirements.md`
>
> Para que el sistema orquestador y los agentes de la agencia de desarrollo procesen los requisitos correctamente, asegúrese de ejecutar la fase de descubrimiento (`wf-discovery`) para que el CEO genere dicho archivo en la carpeta del proyecto y lo marque con la frase `"Entrevista validada"`.
